3teneSTUDIO

このサイトでは VTuber 用ソフト
3tene (ミテネ) の機能紹介をしています。

最新版 3tene V4 の各機能の使い方も随時追加していきます。
3tene FREE は Windows もしくは macOS で無料で使えますので、ぜひお試しください！

・3tene V2 の使い方は旧バージョンの使い方を参照してください。
・3tene V3 の使い方は旧バージョンの使い方を参照してください。

・3tene v4 で OBS の映像キャプチャを使う場合は
　ウェブカメラ一覧で「3tene Screen Capture」を選択してください。
　※OBS Ver 29.1.0 以降で映像キャプチャを使う場合は3tene v3.0.8 以降が必要です。
　※Steam 版で利用するには手動登録が必要になります。

・LeapMotion が動かない場合はソフトウェアのバージョンを確認してください。
・Windows11 で「一部がブロックされています」が表示される場合は
　「スマートアプリコントロール」の設定を確認してください。
